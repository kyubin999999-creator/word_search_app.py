# Word Search App

이 앱은 사용자가 입력한 단어에 대해 관련 이미지를 검색하고, 그 단어의 설명을 제공합니다. 

## 사용 방법

1. 단어를 입력하세요.
2. 해당 단어에 대한 이미지와 설명이 나타납니다.

## 설치 방법

1. 이 프로젝트를 클론합니다.
    ```bash
    git clone https://github.com/yourusername/word_search_app.git
    ```
2. 필요한 라이브러리를 설치합니다.
    ```bash
    pip install -r requirements.txt
    ```
3. Streamlit 앱을 실행합니다.
    ```bash
    streamlit run word_search_app.py
    ```
